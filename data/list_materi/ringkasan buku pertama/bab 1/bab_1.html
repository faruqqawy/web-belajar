Pengertian HTML

HTML atau HyperText Markup Language merupakan bahasa markup yang digunakan untuk membuat halaman website yang dapat ditampilkan melalui web browser.

HTML versi terbaru adalah HTML5. Versi ini tidak hanya mendukung teks dan gambar, tetapi juga elemen interaktif seperti menu, audio, video, dan berbagai fitur lainnya.

HTML termasuk bahasa yang relatif mudah dipelajari dibandingkan dengan bahasa pemrograman lainnya, sehingga sering menjadi dasar dalam pembelajaran pembuatan website.

Sejarah HTML

HTML pertama kali diperkenalkan pada tahun 1989 oleh Tim Berners-Lee. Bahasa ini kemudian dikembangkan oleh organisasi World Wide Web Consortium (W3C).

Pada tahun 2004 dibentuk organisasi WHATWG (Web Hypertext Application Technology Working Group) yang juga berperan dalam pengembangan HTML hingga saat ini.

Perkembangan HTML

  • 24 November 1995 – HTML 2.0 dipublikasikan sebagai IETF RFC 1866 dengan fitur:
    • Upload file melalui form
    • Tabel
    • Peta gambar berbasis klien
    • Dukungan internasional
  • 14 Januari 1997 – HTML 3.2 dipublikasikan oleh W3C.
  • 18 Desember 1997 – HTML 4.0 dirilis dengan tiga variasi:
    • Strict (ketat)
    • Transitional
    • Frameset
  • 24 Desember 1999 – HTML 4.01 (perbaikan minor dari HTML 4.0).
  • 15 Mei 2000 – ISO HTML berdasarkan HTML 4.01 Strict.
  • 28 Oktober 2014 – HTML5 menjadi rekomendasi resmi W3C.

Fungsi HTML

Dokumen HTML mirip dengan dokumen teks biasa, tetapi di dalamnya terdapat kode khusus yang disebut tag. Tag digunakan untuk menentukan struktur dan format halaman web.

HTML berfungsi untuk menyusun komponen halaman web seperti teks, gambar, tautan, dan elemen lainnya. Browser kemudian menerjemahkan kode HTML menjadi tampilan yang dapat dibaca pengguna.

Informasi lebih lanjut dapat dibaca di: https://id.wikipedia.org/wiki/HTML

Beberapa hal yang dapat dilakukan dengan HTML:

  • Membuat judul dan heading.
  • Menampilkan teks tebal atau miring.
  • Mengubah ukuran tulisan.
  • Menggabungkan teks dengan gambar.
  • Membuat tautan (link).
  • Menambahkan audio dan video.
  • Membuat formulir interaktif.

Web Browser

Web browser adalah aplikasi yang digunakan untuk mengakses dan menampilkan halaman website di internet.

Browser membaca dokumen HTML lalu menampilkan hasilnya dalam bentuk tampilan visual yang dapat dipahami pengguna.

Menulis HTML di Notepad atau TextEdit

Saat belajar HTML, disarankan menggunakan editor teks sederhana seperti Notepad atau TextEdit. Dengan cara ini, kita dapat memahami struktur kode HTML dengan lebih jelas sebelum menggunakan editor profesional.

Dasar-Dasar HTML

  • Dokumen HTML

    Dokumen HTML dimulai dengan deklarasi <!DOCTYPE html>.

    Struktur dasar HTML:

    <html> ... </html>

    Bagian yang ditampilkan pada halaman web berada di dalam tag <body>.

  • Judul (Heading)

    Judul dalam HTML ditentukan dengan tag <h1> hingga <h6>. Tag h1 adalah yang paling penting, sedangkan h6 yang paling kecil.

  • Paragraf

    Paragraf dibuat menggunakan tag <p>.

  • Tautan (Link)

    Tautan dibuat dengan tag <a> dan menggunakan atribut href untuk menentukan alamat tujuan.

  • Gambar

    Gambar ditampilkan menggunakan tag <img> dengan atribut seperti:

    • src (sumber gambar)
    • alt (teks alternatif)
    • width dan height (ukuran gambar)

  • Tombol

    Tombol dibuat menggunakan tag <button>.

  • Daftar (List)

    Daftar dapat dibuat dengan:

    • <ul> (unordered list)
    • <ol> (ordered list)
    • <li> (list item)

  • Elemen Kosong

    Elemen kosong adalah elemen HTML tanpa isi. Contohnya <br> yang berfungsi untuk membuat baris baru.

Struktur elemen HTML biasanya dapat memiliki atribut tambahan, misalnya pada tag <img>.
