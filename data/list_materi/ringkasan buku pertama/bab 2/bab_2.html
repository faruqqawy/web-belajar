daftar beberapa atribut :

Atribut Penjelasan
alt menentukan teks alternatif untuk gambar, ketika gambar tak dapat ditampilkan
disabled menentukan bahwa elemen input harus dinonaktifkan
href menentukan URL (alamat web) untuk tautan
id menentukan ID unik untuk suatu elemen
src menentukan URL (alamat web) untuk gambar
style menentukan gaya CSS sebaris untuk suatu elemen
title menentukan informasi tambahan tentang elemen (ditampilkan sebagai ujung alat)

Atribut href

tautan HTMl didefinisikan dengan tag <a>. Alamat tautan ditentukan dalam atribut href :

<a href="https://www.google.com"> ini adalah google </a>

maka, pada browser akan tampil:

Atribut src

nama file sumber gambar ditentukan dalam atribut src:

<img src="img_apple.jpg">

jika file gambar berada di folder lain, harus disesuaikan:

<img src="folder/img_apple.jpg">
